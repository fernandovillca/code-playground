## Caracteristicas

- `Inyeccion de dependencia:` Los componentes se pueden inyectar unos a otros (para reutilizar y desacoplar el componente)
- `MVC:` En una API REST la vista seria el JSON
- `API REST FULL`
- `Programacion Reactiva WebFlux`
- `Programacion orientada a aspectos AOP`
- `jdk minimo 17`
- `Spring Segurity JWT`
- `Spring Data Jpa, Validacion, Seguridad, JUnit5, Microservicios, Cloud`

## Que es Spring Boot

- Crea aplicacaiones con Spring Framework, Rest Full, MVC, IoC, Data Access
- Incluye un servidor embebido Tomcat 10
- Versiones de dependencia son auto-administradas
- Auto configuracion de clases proveidas por defecto
- Herramienta de desarrollo devtools
- Monitoreo y metricas para produccion, auditing, health
- Listo para produccion, llegar y ejecutar, jar ejecutables (no requiere war)
